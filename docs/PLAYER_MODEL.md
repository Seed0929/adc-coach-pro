# Player Model

Version: 1.0

---

# Purpose

The Player Model is BotDiff's long-term memory.

Rather than evaluating matches independently, BotDiff continuously builds a profile of each player's strengths, weaknesses, habits, and improvement trends.

Every analyzed game updates the Player Model.

The Player Model makes future coaching more personalized.

---

# Coaching Philosophy

One match can be misleading.

Ten matches reveal habits.

Fifty matches reveal identity.

BotDiff coaches habits, not isolated mistakes.

# Habit Tracker

Track recurring habits.

Positive Habits

• Maintains high CS.
• Rarely dies before first item.
• Consistent objective attendance.
• Strong lane trading.

Negative Habits

• Greedy recalls.
• Overextends after winning lane.
• Dies before Baron.
• Chases kills.
• Misses side waves.

Habits become more reliable as more games are analyzed.
