# BotDiff Patch Update Process

Version: 1.0

Status: Active

Priority: High

---

# Purpose

League of Legends evolves every patch.

Champion balance, items, runes, objectives, and meta strategies constantly change.

This document defines the standardized process BotDiff follows whenever Riot releases a new patch to ensure coaching recommendations remain accurate and up-to-date.

---

# Objectives

The Patch Update Process must:

- Detect new Riot patches
- Update champion knowledge
- Update matchup knowledge
- Adjust scoring expectations
- Refresh AI coaching context
- Preserve historical analyses
- Maintain coaching consistency across versions

# Success Criteria

The Patch Update Process succeeds when:

BotDiff reflects new League patches within a short time after release.

Champion guides remain accurate.

Matchup advice remains relevant.

AI coaching references the current meta.

Historical reports remain consistent with the patch they were generated on.

Players receive coaching that reflects today's game—not last month's.
