# 09 — Future Features (Backlog)

A living backlog of planned features. Not commitments to a specific version —
see 02_Product_Roadmap for sequencing. Documentation only.

## Experience & UI
- Four UI themes (Classic Dark, Challenger Blue, Neon, Frost/Glass). *(shipped)*
- Coaching timeline.
- Progress tracker.
- Achievement system.

## AI & Coaching
- OpenAI integration (live coaching via the AI Provider layer).
- Voice coaching.
- Challenger Perspective (how a Challenger would have played the moment).
- Goal engine (streak-based focus rotation).

## Riot & Data
- Live Riot sync (near-real-time after games).
- Replay This Match (replay-driven coaching).

## Strategy Coaching
- Draft Coach.
- Champion matchup coach.
- Team composition analysis.
- Best build recommendations.
- Lane plans.

## Social & Scale
- Tournament mode.
- Team coaching.
- Expansion to every League role (Support, Mid, Jungle, Top) with ADC as the
  flagship reference implementation.

## Notes

Every backlog item must reuse the shared coaching engine, memory model, Master
Coaching Prompt, and design tokens. New features never fork the architecture.
