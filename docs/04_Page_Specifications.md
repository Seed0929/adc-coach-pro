# 04 — Page Specifications

Each page has one primary action and never feels cluttered. Every statistic
must explain why it matters.

## Dashboard
- **Purpose:** the player's home; snapshot of current focus and recent results.
- **Main widgets:** current practice goal, latest grade, Analyze Match CTA.
- **Secondary widgets:** recent matches, average grade, progress trend.
- **Future:** proactive "since your last sync" briefing, streak indicators.

## Profile
- **Purpose:** long-term identity and skill profile.
- **Main widgets:** overall skill grades, current coaching focus, biggest
  strength/leak.
- **Secondary widgets:** champion pool, playstyle identity, confidence scores.
- **Future:** identity evolution timeline, per-champion drill-down.

## AI Coach
- **Purpose:** conversational, personalized coaching.
- **Main widgets:** proactive briefing, Quick Ask, structured answer.
- **Secondary widgets:** follow-up questions, next-game challenge.
- **Future:** live AI streaming, voice coaching, replay-linked answers.

## Coaching
- **Purpose:** structured coaching library derived from the dossier.
- **Main widgets:** weakness patterns, strength patterns, active focus.
- **Secondary widgets:** habit rates, evidence citations.
- **Future:** matchup library, drill catalog.

## Match History
- **Purpose:** browse imported ranked games.
- **Main widgets:** match cards (champion, result, KDA, CS, duration, queue).
- **Secondary widgets:** filters, analyze button per match.
- **Future:** search, tags, per-match evidence preview.

## Champion Analysis
- **Purpose:** per-champion performance and coaching.
- **Main widgets:** games, win rate, average grade, best/weakest skill.
- **Secondary widgets:** trend, matchup notes.
- **Future:** champion mastery tracking, build/rune recommendations.

## Analytics
- **Purpose:** progress and trends over time.
- **Main widgets:** grade-over-time graph, consistency, improvement timeline.
- **Secondary widgets:** last 5/10/25/50 windows.
- **Future:** predictive climb estimate, comparison to rank benchmarks.

## Settings
- **Purpose:** account and appearance control.
- **Main widgets:** profile, connected Riot account, theme picker.
- **Secondary widgets:** notifications, account management.
- **Future:** coaching preferences, focus overrides.

## Premium Dashboard
- **Purpose:** the "hired a coach" home for members.
- **Main widgets:** daily coaching, weekly report, active goals.
- **Secondary widgets:** coaching timeline, progress history.
- **Future:** replay coaching queue, matchup library shortcuts.

## Match Review
- **Purpose:** deep single-match coaching report.
- **Main widgets:** overall grade, biggest strength, biggest leak, evidence,
  practice goal.
- **Secondary widgets:** category grades, coach's notes.
- **Future:** "Replay This Match", Challenger Perspective overlay.
