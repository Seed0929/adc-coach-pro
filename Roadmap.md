# BotDiff MVP Specification

Version: 1.0

---

# Mission

BotDiff helps ADC players improve through personalized AI coaching based on their own ranked matches.

The MVP focuses on one role only:

ADC.

Nothing else.

---

# Target User

League of Legends players.

Rank:

Iron → Emerald

Primary Goal:

Improve faster through personalized coaching.

---

# Core User Flow

1. Create an account.
2. Connect Riot account.
3. Sync recent ranked matches.
4. Select a match.
5. Click "Analyze Match."
6. Receive an AI coaching report.
7. Complete a practice goal.
8. Repeat.

---

# MVP Features

## Authentication

- Sign up
- Sign in
- Sign out
- Protected dashboard

---

## Riot Integration

- Connect Riot ID
- Store PUUID
- Sync recent ranked games
- Refresh match history

---

## Match History

Display:

- Champion
- Result
- KDA
- CS
- Match duration
- Queue
- Date played

---

## Match Analysis

Every report includes:

- Overall Grade (F–S+)
- Biggest Strength
- Biggest Leak
- Match Summary
- One Practice Goal

---

## Player Profile

Display:

- Current coaching focus
- Strongest skill
- Weakest skill
- Recent grades
- Progress over time

---

## Dashboard

Display:

- Recent matches
- Latest report
- Current practice goal
- Average grade
- Analyze Match button

---

# Features NOT Included in MVP

- Jungle coaching
- Mid coaching
- Top coaching
- Support coaching
- Live game coaching
- Draft analysis
- Duo coaching
- Clash analysis
- Mobile app
- Discord bot
- Voice coaching

---

# Success Metrics

A successful MVP allows a player to:

- Sign in
- Connect Riot
- Analyze a match
- Learn one actionable improvement
- Return for another analysis

---

# Definition of Done

BotDiff is ready for beta when a user can complete the full coaching flow without developer assistance.
